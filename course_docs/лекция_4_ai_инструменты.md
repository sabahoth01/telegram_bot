# Лекция — AI-инструменты для разработки

---

## Обзор инструментов (разбиралось на занятии)

На занятии сравнивались современные AI-инструменты для разработчиков: GPT, Claude, Qwen, Cursor, Claude Code, Gemma.

---

## Языковые модели (LLM)

### GPT-4 / ChatGPT (OpenAI)
- Сильный универсальный помощник
- С режимом browsing — ходит в интернет
- Платный (есть бесплатный уровень с ограничениями)
- Доступен через API

### Claude (Anthropic)
- Очень длинный контекст (до 200K токенов)
- Сильное рассуждение и анализ длинных документов
- Есть инструменты (tools/functions): поиск в интернете, выполнение кода
- Claude Code — отдельный агентский CLI-инструмент
- Модели: Claude Opus (мощный), Claude Sonnet (баланс), Claude Haiku (быстрый/дешёвый)

### Qwen (Alibaba)
- Хорошая многоязычная поддержка (особенно китайский)
- Открытые версии доступны через HuggingFace
- Несколько размеров моделей

### Gemma (Google DeepMind)
- **Открытая модель** (open-source, бесплатная)
- Варианты: 2B и 7B параметров
- Запускается локально через **Ollama**
- Не ходит в интернет (работает офлайн)
- Подходит для приватных задач

## AI-инструменты для разработки

### Cursor
- IDE на базе VSCode с встроенным AI
- Понимает контекст всего проекта (не только текущего файла)
- Может использовать разные модели под капотом (GPT-4, Claude и др.)
- Доступ в интернет — через выбранную модель
- Умеет: автодополнение, рефакторинг, объяснение кода, генерация по описанию

### Claude Code
- Агентский CLI-инструмент от Anthropic
- Работает в терминале
- Может: читать файлы, писать код, запускать команды, делать git-операции
- Доступ в интернет через инструменты модели
- Особенность: агентский режим — сам принимает решения о следующих шагах

### VSCode с плагинами
- **GitHub Copilot** — автодополнение кода (платный)
- **Continue** — open-source альтернатива Copilot
- **Codeium** — бесплатный AI-помощник
- Доступ в интернет — зависит от плагина

### Gemma локально (через Ollama)
- Полностью офлайн
- Никуда не ходит в интернет
- Приватно: данные не покидают машину
- Запуск:
```bash
ollama pull gemma:2b
ollama run gemma:2b
```

## Вывод (из обсуждения на занятии)

- Для задач, требующих **приватности** или **работы офлайн** — Gemma локально
- Для **продуктивной разработки** с пониманием контекста проекта — Cursor или Claude Code
- Для **разовых вопросов** и объяснений — ChatGPT / Claude в браузере
- Для **встраивания в IDE** без смены редактора — VSCode + плагины
- Все инструменты полезны, выбор зависит от задачи

## Как запустить Gemma локально

```bash
# Установить Ollama
curl -fsSL https://ollama.com/install.sh | sh

# Скачать модель
ollama pull gemma:2b   # лёгкая, ~1.5 ГБ
ollama pull gemma:7b   # умнее, ~4.5 ГБ

# Запустить в терминале
ollama run gemma:2b

# API доступен на
# http://localhost:11434
```
